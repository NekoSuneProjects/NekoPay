Merchant Site Example

NekoPay Hosted Checkout Demo

This example simulates a merchant website creating a checkout session on your NekoPay server, then redirecting the customer to the hosted checkout page.

Use the NekoPay store secret key from the dashboard API key section. Do not use PayPal client ID, PayPal secret, Stripe secret, or webhook secrets here.

What It Does

When you click the button, this page sends a `POST` request to: 

POST /api/merchant/checkout-sessions

With this auth header: 

Authorization: Bearer sk_live_...

If the request succeeds, it redirects the browser to: 

session.checkoutUrl
